# LLL Music

A modern music streaming platform.

## Features
- 🎵 Music streaming & playback
- 📤 Multi-user music uploads
- 🛡️ Admin management dashboard
- 📱 Responsive design for all devices

## Setup

### 1. Backend Setup
```bash
cd server
npm install
npm start
```

### 2. Frontend Setup
```bash
cd client
npm install
npm run dev
```
